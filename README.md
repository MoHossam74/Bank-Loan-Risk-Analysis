# Bank Loan Risk Analysis Dashboard

## SQL Analysis
In this project, SQL Server was used to explore and analyze the bank loan dataset.  
The following analysis was performed using SQL queries:

- Calculated the **total number of loans** in the dataset.
- Measured the **default rate percentage**.
- Analyzed **loan default rate by credit category**.
- Analyzed **loan default rate by debt-to-income (DTI) category**.
- Identified **high-risk customer segments** based on financial characteristics.

---

## Power BI Dashboard
An interactive Power BI dashboard was built to visualize the loan risk analysis.

The dashboard includes:

- **Total Loans KPI**
- **Defaulted Customers KPI**
- **Default Rate Percentage**
- **Average Risk Score**
- **Default Rate by Credit Category**
- **Default Rate by DTI Category**
- **Loan Default Risk Heatmap**
- **Top Risk Segments by Default Rate**

The dashboard allows users to quickly identify **high-risk borrowers and financial risk patterns**.
